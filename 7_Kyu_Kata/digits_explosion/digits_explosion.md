## Kata Description


Given a string made of digits `[0-9]`, return a string where each digit is repeated a number of times equals to its value. 



## Examples



```python

explode("312")

```



should return :



```python

"333122"

```

___

```python

explode("102269")

```



should return :

```python

"12222666666999999999"

```
